Initial Server Setup with Rocky-10

A detailed guide to viewing detailed file information in the Linux terminal.

Step-by-Step Executions

Update repositories,

corporealsoul@localhost:~$ sudo yum update -y

Name assigned to the machine on the network,

corporealsoul@localhost:~$ sudo vi /etc/hostname
corporealsoul@localhost:~$ cat /etc/hostname 
corporealsoul-server

Resolve hostnames without the need for a DNS server,

corporealsoul@localhost:~$ sudo vi /etc/hosts
corporealsoul@localhost:~$ cat /etc/hosts 
192.168.56.4 corporealsoul-server

Configure Static IP,

corporealsoul@localhost:~$ sudo nmtui 
Static Address : 192.168.56.4 
Gateway : 192.168.56.1 
DNS Server : 8.8.8.8, 8.8.4.4

Customizing the Shell Environment - User,

corporealsoul@corporealsoul-server:~$ vi .bashrc 
export PS1="\[\e[01;37m\][\[\e[0m\]\[\e[01;32m\]\u\[\e[0m\]\[\e[00;37m\]@\[\e[0m\]\[\e[01;34m\]\h\[\e[0m\]\[\e[00;37m\] \[\e[0m\]\[\e[00;37m\]\t\[\e[0m\]\[\e[01;37m\] \W]\\$ \[\e[0m\]"
corporealsoul@corporealsoul-server:~$ source ~/.bashrc

Customizing the Shell Environment - Root,

corporealsoul@corporealsoul-server:~$ sudo su -
root@corporealsoul-server:~# vi .bashrc 
export PS1="\[\e[01;37m\][\[\e[0m\]\[\e[01;31m\]\u\[\e[0m\]\[\e[00;37m\]@\[\e[0m\]\[\e[01;34m\]\h\[\e[0m\]\[\e[00;37m\] \[\e[0m\]\[\e[00;37m\]\t\[\e[0m\]\[\e[01;37m\] \W]\\$ \[\e[0m\]"
root@corporealsoul-server:~# source ~/.bashrc

Generate a new SSH key pair,

[corporealsoul@corporealsoul-server 16:18:27 ~]$ ssh-keygen -t ed25519
[corporealsoul@corporealsoul-server 16:18:27 ~]$ ssh-copy-id corporealsoul@192.168.56.4
[corporealsoul@corporealsoul-server 16:18:27 ~]$ ssh-copy-id corporealsoul@192.168.56.8
[corporealsoul@corporealsoul-server 16:18:27 ~]$ cat ./.ssh/authorized_keys

Elevated privileges (root access) without requiring a password,

[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo visudo 
corporealsoul ALL=(ALL) NOPASSWD:ALL

Required Software Installation,

[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo yum update -y
[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo yum install epel-release -y
[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo yum install htop -y
[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo yum install make -y
[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo yum install git -y
[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo yum install curl wget -y
[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo yum install net-tools -y

Manage Firewall,

[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo systemctl status firewalld.service
[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo systemctl enable --now sshd
[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo systemctl enable --now firewalld
[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo firewall-cmd --permanent --add-service=ssh
[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo firewall-cmd --permanent --add-service=http
[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo firewall-cmd --permanent --add-service=https
[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo firewall-cmd --reload

Manage SELinux,

[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo setenforce 1
[corporealsoul@corporealsoul-server 16:18:27 ~]$ sudo vi /etc/selinux/config 
# Set SELINUX=enforcing
