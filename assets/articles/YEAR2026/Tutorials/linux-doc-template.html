Command Documentation: ls -l

A detailed guide to viewing detailed file information in the Linux terminal.

1. Overview and Purpose

The **ls** command is used to list files. The **-l** flag modifies the output to display results in a long format, providing permissions, ownership, size, and timestamp information.

Syntax

ls [OPTIONS] [FILE]...

2. Prerequisites

  • Access to a Linux terminal (Bash or Zsh).
  • Basic understanding of file permissions.

3. Step-by-Step Execution

Step 3.1: Execute the Basic Long Listing

Run the command in your home directory (~) to see the files and directories.

$ ls -l

Step 3.2: Analyze the Output

The result is a table-like structure. The output block below shows an example. Notice the leading character (d for directory, - for file) and the **permissions string**.
